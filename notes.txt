HOW NODE WORKS 
+---------+       +-----------+       +----------+
| Browser | --->  | HTTP Server| ---> |  Handler |
+---------+       +-----------+       +----------+
                                      | Business |
                                      |  Logic   |
                                      +----------+
Browser (client) makes a request (e.g. GET request).
Node.js HTTP Server listens to request (using http.createServer()).
Passes request to your handler function.
You send back response.
